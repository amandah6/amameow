Welcome to my Website!

Hello! Welcome to my website :3 Here you’ll find all the things that I love, like sewing, crafts, and fixing things! I also collect bottlecaps.

I no longer actively use social media because it stresses me out! But I still feel like it’d be cool to have a place to record what I’ve been up to and share it with people, so this is what this website is.

About this site

This site was hand-coded by me in HTML, CSS, JS, and a bit of golang templates. It is responsive and is designed to be viewed on any screen resolution. Built using Hugo, (a Static Site Generator)! Currently hosted by neocities <3

Summary of Pages

  • Bottlecaps: browse my bottlecap collection!

  • Projects: blog-style updates to what I’m working on now. Each post is tagged, allowing you to sort by topics like sewing, computers, crafts, or whatever random stuff I’m writing about.

  • Recipes: new stuff I’ve baked or cooked, with pictures if I have them!

  • Links: links to web resources I used to make my site + button wall
